Basic Rules of Ludo game:


→  A token can start to maneuver as long as the dice rolled maybe a 6.

→  Each player gets a turn wise chance to roll the dice. And if the player rolls a 6, they're going to get another chance to roll the dice again.

→  All the tokens must reach the middle of the board to win the game.

→  The token moves clockwise consistent with the number of rolled dice.

→  Knocking out another's token will offer you an extra chance to roll the dice again.


How the Ludo Jungle Game Works:
