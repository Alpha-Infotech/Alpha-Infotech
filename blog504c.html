The new Android 10 features you should know about!

Google ditched the dessert-themed naming scheme when Android 10 was released. However, the lackluster two-year major updates and three-year security updates on smartphones remain.

Not all Android smartphones will receive the Android 10 update especially devices that are on the budget category. For mid-range smartphones that are two years or newer, there's still an opportunity. Here’s a list of smartphones receiving Android 10 anytime soon.

For those that will have their update soon or if you have already got the update but haven't explored, here are the top 10 features everybody should be using.

1. Gesture Navigation

This new feature on Android 10 allows the user to navigate between apps through swipes and taps. Google’s implementation is neat, but it takes a while to get used to it after the update.

There will vary methods in activating Gesture navigation counting on the brand. On Nokia devices, it activates once the update is installed. generally, here are the steps to activate gesture navigation supported Pixel 4.

Open Settings

Systems

Gestures

System navigation

Select Gesture navigation

2. Personal Wi-Fi QR codes

Android 10 added a new WiFi feature for smartphones. It allows users to make a QR code for their Wi-Fi network. It also allows users to attach to a Wi-Fi network through QR codes. This feature is convenient for personal networks with tons of incoming visitors. they will easily connect with the network without typing a password.

To activate the feature:

 Open Wi-Fi and Network settings

Choose the specified network

Click the share icon and a QR code will appear

3. Dark mode

The Dark mode might seem sort of a feature that's already present in previous Android versions. But it had been only this time that Google made it official. The previous dark mode was experimental and it only activates on certain Google apps. This improved dark mode is system-wide, which allows most apps to possess a darker theme. Even the wallpaper gets dimmer once it activates.

Activating the dark mode takes two simple steps. Just open the Display settings then toggle the Dark mode settings.

Using the dark mode allows for extended battery life, especially for devices with OLED display. for a few smartphones, the dark mode also activates when the battery saver is on.

4. Smart reply to messages

When it involves communication, this is often one of Google’s best features so far. The Smart reply predicts the most effective reply to a message received on any messaging app.

Whenever there's a message notification on Facebook Messenger, Google Chat, Telegram, or WhatsApp, the smart reply exposes on the notification bar and allows for short quick replies. This feature is great for users who are in a meeting or public transportation.

The only downside is, if the user lives in a non-English speaking country, the smart reply won't activate.

5. Focus mode

Smartphones are undoubtedly used for productivity nowadays. This new feature helps users whose main business or work involves the usage of smartphones. The new Focus mode is part of Google’s Digital Wellbeing suite.

When activated, the Focus-mode disables notifications from apps that the system detects as a distraction while working.

Another feature added to the Digital Wellbeing suite is that the parental controls. this allows parents to watch their child’s activities or prevent them from doing certain tasks.

6. Undo app removal

There will always be an instance wherein a user mistakenly removes an app from the home screen accidentally. Once an app is removed, sometimes it gets hard to recollect what it had been or where was the first location.

This little feature gives the user a little bit longer to undo accidental changes on the house screen. If the remove app option gets activated, there'll be an undo app changes option at the bottom of the screen.

7. Improved privacy settings

The Android 10 brings a dedicated Privacy option on the Settings menu. it's improved permissions for location, camera, files, contacts, and other private concerns.

The new section allows the user to possess a transparent view as to what apps have access to a particular setting. It also provides a better way of disabling app access to non-public data.

With the recent privacy issues surfacing like Zoom’s access to data, this Android 10 feature provides more control with the user’s privacy settings.

8. Live captions

The Live Captions feature gives deaf and people with impaired hearing a better way to navigate the smartphone. This feature allows Google Assistant to process speech with no delay. It is an on-the-spot caption for everything inside the smartphone.

The great thing with this feature is it doesn't need mobile data to work. Everything happens offline, which makes it easier to use during emergencies. The feature is accessible through the accessibility settings.

9. Improved notification controls

Notifications are intrusive sometimes. With Android 10, the notification bar now offers the choice to “Alert” or “Silent” an app notification.

When choosing alert, the app continues to create a sound sort of a regular notification. The silent, because the name implies, disable sound or vibration.

This is a small feature that may seem useless to other users. However, its main usage is to customize the smartphone to react differently on each app unlike the vibrate or silent option through the quantity rocker.

This feature is accessed within the Notifications tab under the Apps & Notifications settings.

10. Location control

In previous Android versions, the placement access is usually allowed once set on the primary try. This time, Android 10 gives a more flexible option on how apps use location information.

Users now have the choice of letting an app access the situation data whenever an app is open. Once the app isn't in use, location access is going to be disabled. This feature saves battery life also.
